## 🎯 Objectives
- Analyze sales across regions and platforms
- Identify top-performing games and genres
- Understand market trends over time

## 📊 Tools Used
- Power BI
- Excel

## 📁 Dataset
- Video Game Sales Dataset (Kaggle)

## 🔍 Key Insights
- Action and Shooter genres dominate sales
- Few publishers control a major share
- Sales vary significantly by region
- Platform popularity affects performance
